# SpeedEdTech Website Redesign

A modern and fully responsive redesign of the SpeedEdTech website built using HTML, SCSS, and JavaScript.  
This project focuses on creating a clean, professional, and user-friendly interface with improved UI/UX and smooth interactions.

## Features
- Modern and responsive design
- Clean and organized layout
- Interactive UI elements
- Mobile-friendly interface
- Smooth animations and transitions
- Optimized user experience
- Reusable SCSS structure

## Technologies Used
- HTML5
- SCSS (Sass)
- JavaScript

## Project Goal
The main goal of this redesign was to improve the overall look and feel of the website while making the user experience smoother, more modern, and visually appealing.
